# Skill: Architecture RFC Steward

Use this skill when a task changes the long-term Navfolio architecture.

## Responsibilities

- Preserve the core plus plugin ecosystem direction from issue #68.
- Keep package boundaries explicit.
- Identify decisions that need owner confirmation.
- Update plans before implementing broad refactors.

## Checklist

- Does this change keep core independent from theme UI?
- Does this change make plugin contracts clearer?
- Does this change reduce monolithic coupling?
- Does this change need a migration note?
- Does this change align with the Quartz-style core/community split?
